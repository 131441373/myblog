### 空间分配
#### operator new/delete
是类内自己定义 的，如`A::operator new`
c++还有个默认的`::operator new`

#### new/delete operator
new = operator new + constructor
delete = destructor + operator delete

#### placement new
在一个指定的地址创建对象并返回指针

#### example
```c++
void* place = ::operator new(sizeof(int));
int *p = new(place) int(1);
//int *p = new int(1);
```
